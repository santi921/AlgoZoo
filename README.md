# CancerImaging


This is a module that aims to facilite the discovery of effective algorithms for detecting malaties in low-resolution medical images. The dependencies for this include:

- Python 3.5 
- Numpy 
- joblib 0.13.2
- h5py 2.9.0
- pandas 0.24.2
- Pillow 6.0.0
- Augmentor 0.2.3
- xgboost 0.90
- tensorflow-gpu 1.14.0/ tensorflow 
- scipy 1.3.0
